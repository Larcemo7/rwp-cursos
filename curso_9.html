Problemas com as impressoras? vamos ver como podemos resolver

Se o problema foi erro ao imprimir, pode ser um simples problema de mal funcionamento da impressora.

1 → Verifique se não esta sem papel, uma vez que ela alerta (para alguns modelos) basta colocar o papel, abrindo a impressora que suas funções devem normarlizar (siga a orientação correta do papel, para não coloca - lo errado)

2 → Se ela for via USB, e os drives estiverem devidamente instalados, é plaúsivel verificar o cabo USB, uma vez que a conexão for via rede, vai ser necessario alguns passos extras para verificar se esta ok.

Se a conexão for via rede, vai ser necesario algumas etapas extras. Vá nas configurações do Windows:

Se a impressora estiver instalado, basta verificar se o IP esta correto, para verificar o IP da impressora, com ela desligada, pressione o botão de "papel" (que pode varias para cada modelos) e ligue ela, continue pressionando o botão de papel por alguns segundos (varia de acordo com os modelos)

Pronto, agora busque no papel que saiu o termo: Ip Address: 192.168.??.?? , sendo o simbolo de interrogação os valores da impressora. OBS: não confunda com outros valores, somente os valores depois de IP Address que são relevantes aqui
